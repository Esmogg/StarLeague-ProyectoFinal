¿Que es?

Starleague es un Sistema de Gestión Deportiva Modular (SGDM), una plataforma web orientada a la organización, administración y seguimiento de competencias deportivas, mentales y electrónicas, El sistema permite gestionar torneos de diferentes tipos desde una única pagina, evitando el uso de soluciones separadas o plantillas manuales para cada tipo de competencia.

El sistema se desarrollara utilizando PHP para el backend, HTML, CSS y JavaScript para el frontend, y MySQL como gestor de base de datos relacional. La arquitectura seguirá el patrón Modelo-Vista.Controlador (MVC)

¿Cuales son las funcionalidades?

El SGDM contempla tres tipos de competencia: liga (todos contra todos con acumulacion de puntos), eliminación directa (el perdedor queda eliminado) y sistema suizo (emparejamiento por rendimiento acumulado). Cada formato tiene su propia lógica de generación de enfrentamiento y cálculo de posiciones, todo automatizado por el sistema.

El sistema estará orientado a clubes deportivos, instituciones educativas, organizadores independientes y comunidades de deportes electrónicos o de mesa, permitiendo administrar participantes, rondas, resultados y tabla de posiciones desde el panel centralizado.
